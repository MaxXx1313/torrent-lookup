# TLookup

TLookup is a desktop application that scans your filesystem for .torrent files, analyzes their content, and finds the
actual downloaded files associated with each torrent.

## Purpose

The application helps you:

* Locate all .torrent files on your computer
* Find which files have actually been downloaded
* Upload found torrents to Transmission client (currently the only supported client)

## Related Tools

For command-line usage, check out the [TLookup CLI](./lib/README.md) tool.
